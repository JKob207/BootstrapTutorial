Lista rozwijana

Podstawowa lista

Lista może być użyteczna w przypadku gdy chcemy schować wiele elementów, opcji, albo też linków. Możemy w taki sposób je pogrupować i ukryć pod odpowiednim przyciskiem.

Lista w przycisku

Jeśli chcemy stworzyć listę, która będzie ukryta pod przyciskiem, musimy całość jej kodu umieścić w divie z klasą .dropdown. Może to też być dowolny element który posiada właściwość postion: relative. W jego wnętrzu tworzymy button lub znacznik <a> z klasą .dropdown-toggle oraz z atrybutem data-toggle="dropdown". Następnie tworzymy odrębnego diva z klasą .dropdown-menu i w jego wnętrzu umieszczamy nasze linki z klasą .dropdown-item, które będą elementami naszej listy.

Pierwszy element Drugi element Trzeci element 
<div class="dropdown">
    <button class="btn btn-primary dropdown-toggle" type="button" id="listaPodstawowa"
        data-toggle="dropdown" aria-haspopup="true" aria-expanded="false">
        Kliknij!
    </button>
    <div class="dropdown-menu" aria-labelledby="listaPodstawowa">
        <a class="dropdown-item" href="#">Pierwszy element</a>
        <a class="dropdown-item" href="#">Drugi element</a>
        <a class="dropdown-item" href="#">Trzeci element</a>
    </div>
</div>

Lista w rozdzielonym przycisku

Możemy stworzyć także przycisk który ma część odpowiedzialną za wyświetlanie listy. Tworzymy w tym celu naszego diva z klasą .btn-group albo element z pozycją relatywną. W jego wnętrzu umieszamy dwa elementy typu button. Pierwszy z nich stylujemy jak zwykły przycisk. Natomiast drugiemu, czyli to jest ten który rozwija naszą listę, oprócz podstawowych klas nadajemy także klasy: .droppdown-toggle .dropdown-toggle-split. Dodajemy mu także atrybut data-toggle="dropdown". Następnie dodajemy odrębnego diva z klasą .dropdown-menu i w jego wnętrzu umieszczamy nasze linki z klasą .dropdown-item, które będą elementami naszej listy.

Pierwszy element Drugi element Trzeci element 
<div class="btn-group">
    <button class="btn btn-primary" type="button">
        Kliknij!
    </button>
    <button class="btn btn-primary dropdown-toggle dropdown-toggle-split" type="button"
        data-toggle="dropdown" aria-haspopup="true" aria-expanded="false">
    </button>
    <div class="dropdown-menu">
        <a class="dropdown-item" href="#">Pierwszy element</a>
        <a class="dropdown-item" href="#">Drugi element</a>
        <a class="dropdown-item" href="#">Trzeci element</a>
    </div>
</div>

        <a class="dropdown-item" href="#">Pierwszy element</a>
        <a class="dropdown-item" href="#">Drugi element</a>

Lista po prawej

Domyślnie wyświetlanie listy jest wyrównane do lewej strony przycisku. Aby przenieść wyświetlanie na prawą stronę musimy elementowi div z klasą .dropdown-menu dodać klasę .dropdown-menu-right

Pierwszy element Drugi element Trzeci element 
<div class="dropdown">
    <button class="btn btn-primary dropdown-toggle" type="button" id="listaPodstawowaPrawo"
        data-toggle="dropdown" aria-haspopup="true" aria-expanded="false">
        Kliknij!
    </button>
    <div class="dropdown-menu dropdown-menu-right" aria-labelledby="listaPodstawowaPrawo">
        <a class="dropdown-item" href="#">Pierwszy element</a>
        <a class="dropdown-item" href="#">Drugi element</a>
        <a class="dropdown-item" href="#">Trzeci element</a>
    </div>
</div>

Nagłówek listy

Każdej naszej liście rozwijanej możemy nadać jakiś nagłówek. Aby tego dokonać, wewnątrz elementu div z klasą .dropdown-menu, jako pierwszy element listy tworzymy sobie nagłówek "h*" z klasą .dropdown-header

Nagłówek listy
Pierwszy element Drugi element Trzeci element 
<div class="dropdown">
    <button class="btn btn-primary dropdown-toggle" type="button" id="listaPodstawowaNagłówek"
        data-toggle="dropdown" aria-haspopup="true" aria-expanded="false">
        Kliknij!
    </button>
    <div class="dropdown-menu" aria-labelledby="listaPodstawowaNagłówek">
        <h6 class="dropdown-header">Nagłówek listy</h6>
        <a class="dropdown-item" href="#">Pierwszy element</a>
        <a class="dropdown-item" href="#">Drugi element</a>
        <a class="dropdown-item" href="#">Trzeci element</a>
    </div>
</div>

Rozdzielacz

Elementy naszej listy możemy wizualnie rozdzielać. Pomiędzy elementami diva z klasą .dropdown-menu możemy wstawić pustago diva z klasą .dropdown-divider

Pierwszy element Drugi element Trzeci element
Czwarty element 
<div class="dropdown">
    <button class="btn btn-primary dropdown-toggle" type="button" id="listaPodstawowaSeparator"
        data-toggle="dropdown" aria-haspopup="true" aria-expanded="false">
        Kliknij!
    </button>
    <div class="dropdown-menu" aria-labelledby="listaPodstawowaSeparator">
        <a class="dropdown-item" href="#">Pierwszy element</a>
        <a class="dropdown-item" href="#">Drugi element</a>
        <a class="dropdown-item" href="#">Trzeci element</a>
        <div class="dropdown-divider"></div>
        <a class="dropdown-item" href="#">Czwarty element</a>
    </div>
</div>
