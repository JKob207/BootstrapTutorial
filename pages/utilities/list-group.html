Listy

Podstawowy przykład

Standardową liste tworzymy za pomocą znacznika <ul> lub <ol> i umieszczonych w niej znacznikach <li>
Aby je specjalnie ostylować wystarczy, że nadamy elementowi ul lub ol klasę .list-group. Następnie musimy ostylować każdy element li odpowiednio klasą .list-group-item

Linki i przyciski jako lista

Zamiast zwykłych elementów li do budowy naszej listy możemy użyć także linków lub przycisków. Aby stworzyć taką listę tworzymy element ul lub ol i nadajemy mu klasę .list-group. Natomiast elementy li zamieniamy na znaczniki <a> lub <button>. Aby je ostylować musimy użyć klasy .list-group-item, tak samo jak w przypadko zwykłej listy. Dodatkowo jednak musimy dołączyć jeszcze klasę .list-group-item-action. W takich listach możemy również korzystać z klas .active oraz .disabled

    Element 1 Element 2 Element 3 Element 4 Element 5 
<ul class="list-group">
    <a href="#" class="list-group-item list-group-item-action active">Element 1</a>
    <a href="#" class="list-group-item list-group-item-action">Element 2</a>
    <a href="#" class="list-group-item list-group-item-action">Element 3</a>
    <a href="#" class="list-group-item list-group-item-action disabled">Element 4</a>
    <a href="#" class="list-group-item list-group-item-action">Element 5</a>
</ul>

Przykład z użyciem przycisków jako elementów listy: 

<ul class="list-group">
    <button type="button" class="list-group-item list-group-item-action active">Element 1</button>
    <button type="button" class="list-group-item list-group-item-action">Element 2</button>
    <button type="button" class="list-group-item list-group-item-action">Element 3</button>
    <button type="button" class="list-group-item list-group-item-action disabled">Element 4</button>
    <button type="button" class="list-group-item list-group-item-action">Element 5</button>
</ul>

Lista bez ramek

Jeśli nie chcemy aby nasza lista posiadała ramką ją otaczającą możemy w łatwy sposób się jej pozbyć. Aby osiągnąć taki efekt możemy dopisać do naszego znacznika <ul> lub <ol> klasę .list-group-flush. Sposób ten działa nie tylko dla zwykłych list, ale także dla tych zbudowanych z linków lub przycisków.

Lista z odznakami

Do naszej listy możemy dodać odznaki, które mogą posłużyć np. do informowania o ilości postów z danej kategorii. Do zbudowania takiej listy przydadzą się klasy Flexboxa.
Zaczynamy standardowo od utworzenia listy z klasą .list-group oraz dodaniem do niej jej elementów z klasą .list-group-item. Następnie dodajemy do tych elementów trzy następujące klasy:

  • .d-flex - klasa ta pozwala na wyświelanie jako flex ( display: flex; )
  • .justify-content-between - klasa ta rozsuwa zawartość ( justify-content: space-between; )
  • .align-items-center - klasa ta wyśrodkowuje elementy w pionie ( align-items: center;)
Wewnątrz każdego elementu tworzymy element <span> z klasami .badge .badge-primary .badge-pill który wyświeli naszą odznakę.

  • Posty w danej kategorii 21
  • Ilość przedmiotów w koszyku 7 
<ul class="list-group">
    <li class="list-group-item d-flex justify-content-between align-items-center">
        Posty w danej kategorii <span class="badge badge-primary badge-pill">21</span>
    </li>
    <li class="list-group-item d-flex justify-content-between align-items-center">
        Ilość przedmiotów w koszyku <span class="badge badge-primary badge-pill">7</span>
    </li>
</ul>

Lista z rozbudowaną treścią

Dotychczas nasza lista zawierała tylko jeden element. Możemy jednak ją bardziej rozbudować i dodać jej dodatkowe treści czy też inne elementy. Przydatne nam tutaj będą klasy Flexboxa.
Jak zwykle zaczynamy od utworzenia naszej listy z klasą .list-group gdzie umieszamy nasze elementy listy z klasami .list-group-item .list-group-item-action. Do naszych elementów listy musimy dopisać jeszcze klasy:

  • .flex-column - klasa ta ustawia nasze elementy od góry do dołu tworząc kolumnę ( flex-direction: column; )
  • .align-items-start - klasa ta pozwala na ustawienie kolejnych elementów od góry elementu nadrzędnego. ( align-items: flex-start; )
Teraz w tak ostylowanym elemencie klasy możemy stworzyć diva z klasami:
  • .d-flex - klasa ta pozwala na wyświelanie jako flex ( display: flex; )
  • .w-100 - klasa ta powoduje, że każdy element listy ma zajmować całą szerokość elementu nadrzędnego, a każdy kolejny element będzie ustawiał się pod poprzednim. ( width: 100%;)
  • .justify-content-between - klasa ta rozsuwa zawartość ( justify-content: space-between; )
W takim elemencie div możemy teraz umieścić jakiś nagłówek oraz mały dopisek. Poza divem umieszczamy paragraf z tekstem, a na samym dole mały dopisek o kategorii oraz o autorze wpisu.
