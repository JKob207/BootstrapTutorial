Pop-up / Modal

Podstawowy przykład

Wyskakujące okienka, czyli elementy pop-up lub też modale. Używane najczęściej do wyświetlania reklam, w celu przeglądania galerii zdjęć czy też ukazując nam opisy produktów.
Sam kod HTML modala jest rozbudowany i składa się z następujących części:

  • .modal - element nadrzędny całego modala
  • .modal-dialog - element nadrzędny okna modala
  • .modal-content - element nadrzędny zawartości modala
  • .modal-header - element nadrzędny nagłówka modala
  • .modal-title - tytuł w nagłówku modala
  • .modal-body - element nadrzędny treści modala
  • .modal-footer - element nadrzędny stopki modala

</div>

Wyświetlanie modala

Nasz modal jeszcze nam się nie wyświetla. Musimy dopiero stworzyć przycisk który go wywoła. Aby to zrobić nasz modal musi mieć własne id, a przycisk który ma go uruchomić powinien mieć parametry data-toggle="modal" data-target="#przykład1"

<button type="button" class="btn btn-primary" data-toggle="modal" data-target="#przykład1">
    Wyświetl modal
</button>
