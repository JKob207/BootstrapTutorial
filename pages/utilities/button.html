Przyciski

Podstawowe przyciski

Podstawowym przyciskiem jest ostylowany element <button>. Musimy nadać mu dwie klasy: .btn oraz .btn-*, gdzie "*" oznacza kolor przycisku. Możemy mu także dodać klasę .btn-link, wtedy taki przycisk będzie wyświetlany jako zwykły link, ale zachowujący się jak przycisk który ma nadaną własność display: inline-block

<button type="button" class="btn btn-primary">.btn-primary</button>
<button type="button" class="btn btn-secondary">.btn-secondary</button>
<button type="button" class="btn btn-success">.btn-success</button>
<button type="button" class="btn btn-danger">.btn-danger</button>
<button type="button" class="btn btn-warning">.btn-warning</button>
<button type="button" class="btn btn-info">.btn-info</button>
<button type="button" class="btn btn-light">.btn-light</button>
<button type="button" class="btn btn-dark">.btn-dark</button>
<button type="button" class="btn btn-link">.btn-link</button>

Elementy które moga mieć klasę .btn

Nie tylko zwykłego buttona HTML'owego możemy ostylowac za pomocą nadanych klasa. Możemy to także zrobić w przypadku:

  • Znacznika <a> który pełni rolę jako button
  • Przycisku typu submit
  • Inputu typu button
  • Inputu typu submit
  • inputu typu reset
Link 
<a href="#" class="btn btn-primary" role="button">Link</a>
<button class="btn btn-primary" role="submit">Button</button>
<input class="btn btn-primary" type="button" value="Input" />
<input class="btn btn-primary" type="submit" value="Submit" />
<input class="btn btn-primary" type="reset" value="Reset" />

Przyciski z obramowaniem

W Bootstrapie możemy stworzyć także przyciski które nie będą posiadały tła, a jedynie kolorowe obramowanie. Tworzymy je poprzez połączenie klas .btn oraz .btn-outline-*, gdzie "*" oznacza kolor obramowania przycisku.

<button type="button" class="btn btn-outline-primary">.btn-outline-primary</button>
<button type="button" class="btn btn-outline-secondary">.btn-outline-secondary</button>
<button type="button" class="btn btn-outline-success">.btn-outline-success</button>
<button type="button" class="btn btn-outline-danger">.btn-outline-danger</button>
<button type="button" class="btn btn-outline-warning">.btn-outline-warning</button>
<button type="button" class="btn btn-outline-info">.btn-outline-info</button>
<button type="button" class="btn btn-outline-light">.btn-outline-light</button>
<button type="button" class="btn btn-outline-dark">.btn-outline-dark</button>

Rozmiary przycisków

Możemy modyfikować wielkości przycisków. Wystarczy dodać do naszego przycisku klasę dodatkową .btn-lg, która powiększa nasz przycisk lub klasę .btn-sm która zmniejsza nasz przycisk.

<button type="button" class="btn btn-primary btn-sm">Mniejszy przycisk</button>
    <button type="button" class="btn btn-primary">Domyślna wielkość przycisku</button>
    <button type="button" class="btn btn-primary btn-lg">Większy przycisk</button>

Dodatkowo możemy dodać klasę która będzie nam określała, aby nasz przycisk zajął całą dostępną szerokość elementu w którym jest zawarty. Uzyskujemy to dzięki klasie .btn-block. Możemy to łączyć z różnymi rozmiarami przycisków. 

<button type="button" class="btn btn-secondary btn-block btn-sm">Block level small button</button>
<button type="button" class="btn btn-secondary btn-block">Block level default button</button>
<button type="button" class="btn btn-secondary btn-block btn-lg">Block level large button</button>

Aktywny przycisk

Aby nasz przycisk wyglądał na "wciśnięty" możemy użyć klasy .active oraz dodatkowo atrybutu aria-pressed="true"

<button type="button" class="btn btn-warning active" aria-pressed="true">Link</button>

Nieaktywny przycisk

Jeżeli chcemy aby nasz przycisk był nieaktywny i nie było możliwości klikania w niego, możemy użyć atrybutu disabled. Jeżeli chcemy wyłączyć także link musimy użyć w tym przypadku klasy .disabled

Wyłączony link 
<button type="button" class="btn btn-secondary" disabled>Wyłączony przycisk</button>
<a href="#" class="btn btn-secondary disabled" role="button">Wyłączony link</a>
