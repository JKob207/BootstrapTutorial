Paginacja

Podstawowy przykład

Paginacja to element służący do nawigacji. Pozwala na przykład rozbić sporą ilość treści na stronie na kilka pomniejszych podstron, pomiędzy którymi można się przemieszczać.
Paginacje umieszczamy wewnątrz elementu <nav>. Kolejno tworzymy standardową listę <ul> z klasą .pagination. Kolejno dodajemy odpowiednią ilość elementów <li> zależnej od ilości podstron. Nadajemy im klasę .page-item, a poszczególnym linkom do stron nadajemy klasę .page-link

  • 3
  • »
<nav>
    <ul class="pagination">
        <li class="page-item">
            <a class="page-link" href="#">«</div>
        </li>
        <li class="page-item">
            <a class="page-link" href="#">1</div>
        </li>
        <li class="page-item">
            <a class="page-link" href="#">2</div>
        </li>
        <li class="page-item">
            <a class="page-link" href="#">3</div>
        </li>
        <li class="page-item">
            <a class="page-link" href="#">»</div>
        </li>
    </ul>
</nav>

Status paginacji

Możemy rozbudować naszą paginacje o różne statusy. Na przykład gdy jesteśmy na stronie pierwszej to nie potrzebujemy aktywnego linka do wcześniejszych wyników. Gdy natomiast jesteśmy na danej stronie dobrze było by gdyby ta strona była wizualnie odnaczona w paginacji. Uzyskamy takie efekty za pomocą klasy .disabled oraz .active dodanych wraz z klasą .page-item

  • «
  • 1
  • 2
  • 3
  • »
<nav>
    <ul class="pagination">
        <li class="page-item disabled">
            <a class="page-link" href="#">«</div>
        </li>
        <li class="page-item">
            <a class="page-link" href="#">1</div>
        </li>
        <li class="page-item active">
            <a class="page-link" href="#">2</div>
        </li>
        <li class="page-item">
            <a class="page-link" href="#">3</div>
        </li>
        <li class="page-item">
            <a class="page-link" href="#">»</div>
        </li>
    </ul>
</nav>

Położenie paginacji

Domyślnie nasza paginacja układa się po stronie lewej. Korzystając jednak z klas flexboxa możemy ją w łatwy sposób wycentrować lub przesunąć do prawej. W celu wycentrowania dodajemy wraz z klasą .pagination klasę .justify-content-center. Przesunięcie w prawo uzyskamy dodając wraz z klasą .pagination klasę .justify-content-end
